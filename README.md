# ski-resort-status

This is a web page that shows the status for the ski resorts that I care about.

## React + Vite Setup

This project is built with React and Vite.

### Development

To run the project locally:

1. \`npm install\`
2. \`npm run dev\`

### Features

- Real-time (simulated) status for top California ski resorts.
- Premium dark mode design.
- Responsive layout.
